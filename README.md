# 쇼핑 리스트 앱

로컬 브라우저에서 실행되는 간단한 쇼핑 리스트 웹 앱입니다.

## 기능

- 아이템 추가
- 아이템 삭제
- 아이템 체크 (완료 표시)
- localStorage를 사용한 데이터 영구 저장

## 실행 방법

별도의 설치나 빌드 과정이 필요 없습니다. `index.html` 파일을 브라우저에서 직접 열어 사용하세요.

```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
